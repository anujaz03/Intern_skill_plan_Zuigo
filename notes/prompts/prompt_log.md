# Day 2 – Prompt Engineering Log

## Prompt 1 – Core Python Practice

### Goal
Refresh Python fundamentals using a real-world example.

### Prompt

Generate a Python program that manages employee salary records using lists and functions.

Requirements:
- Use at least one function.
- Use a list to store salaries.
- Calculate highest, lowest and average salary.
- Add meaningful comments.
- Follow Python best practices.

### Result

The generated code was clean and modular.
The use of built-in functions like max(), min(), and sum() improved readability.

### What Worked

- Clear goal.
- Specific requirements.
- Requested best practices.
- Real-world example instead of hello-world.

### What Didn't Work

- No input validation.
- No exception handling.
- Could be improved by separating business logic into multiple functions.
