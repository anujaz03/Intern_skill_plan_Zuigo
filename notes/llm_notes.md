### WHAT IS LLM?

-> Large Language Model is an Ai system trained on massive amount of data which can be text or code.
It can understand natural language and generate human like responses.

### What llms are good at?
-> They can generate code in any programming language, generate images, help in debugging , documentation and explain anything.


 ## WHERE LLMS FAIL

 -> They can generate wrong outputs and sometime give wrong answers confidently.
 -> They hallucinate .
 -> They forget the context over long chats.

 ## Conclusion:

 LLMs imporve developer productivity by assisting with coding, debugging , documentation and learning.
 But some how they are not perfect and need to be monitized time to time.
 It is a good practice to check the output and see if it is correct and as expected.
