# Zuigo AI Engineering Training

This repository contains my learning progress during the 14-day AI Engineering Internship at Zuigo. Throughout this program, I worked on AI-assisted development, prompt engineering, product thinking, requirement analysis, UI/UX design, and client communication while building an AI-powered CRM product called LeadFlow AI.

## Project

**LeadFlow AI** is an AI-powered CRM designed for small businesses. The product helps users manage customers, leads, follow-ups, and documents while using AI to organize data, identify missing information, detect duplicates, and recommend follow-up actions.
